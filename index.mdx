## How It Works

Suron Vault encrypts your secrets into a `.env.suron` file that is safe to commit to git. When your server starts, it requests approval via Telegram — you tap **Approve** and the secrets are decrypted directly into `process.env` in RAM.

```javascript
import { vault } from '@suronai/sdk'

await vault.load()   // blocks until you approve on Telegram

console.log(process.env.DATABASE_URL)   // decrypted, RAM only
```

The master key is **split across two independent systems** (Convex + Cloudflare KV). Neither half alone can decrypt anything. Even if one system is compromised, your secrets remain safe.
